# Final Master Plan

This file is the planning source of truth for product phases.

## v1.0 - Supabase PWA

v1.0 is a hosted single-customer, single-center Supabase PWA.

Required scope:
- Real Supabase authentication.
- Real Supabase CRUD for implemented operational modules.
- Browser QA against Supabase mode.
- No Preview Mode as a valid product, demo, fallback, or sales path.

Deferred from v1.0:
- Checkout completion.
- Invoice print.
- Financial reports.
- Settings mutations.
- Expense edit UI.
- Performance polish beyond release blockers.

## v1.1 - Operational Expansion

v1.1 adds verified production support for:
- Checkout.
- Print.
- Reports.
- Settings mutations.
- Expense edit.
- Performance polish.

## v2.0 - Windows Desktop EXE

v2.0 is a future Windows Desktop EXE direction using:
- Tauri v2.
- SQLite.
- Local auth.
- Local backup/export.

Do not implement v2.0 work in documentation cleanup branches.
