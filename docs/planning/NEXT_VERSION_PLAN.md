# Next Version Plan

## Next release after v1.0: v1.1

v1.1 starts only after v1.0 is validated as a real Supabase PWA with browser QA.

Planned v1.1 work:
- Checkout implementation and verification.
- Receipt/invoice print support.
- Sales, revenue, and financial reports.
- Settings mutations and logo/storage behavior.
- Expense edit UI and real Supabase update implementation.
- Performance polish and code splitting.

## Explicit non-goals

- Do not use Preview Mode as a fallback, demo, sales path, or release-verification path.
- Do not position v1.0 as multi-customer SaaS.
- Do not implement Windows Desktop EXE work before v2.0.
