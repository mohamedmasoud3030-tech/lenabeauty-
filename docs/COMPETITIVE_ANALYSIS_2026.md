## 3) ⛔ الفجوات الحرجة (مزايا يمتلكها المنافسون ونفتقدها)

### 🔴 أولوية قصوى — تمنع المنافسة في السوق الخليجي
1. ~~**حساب الضريبة (VAT) في الفاتورة**~~ ✅ **تم التنفيذ (2026-06-28)** —
   migration `20260628000004_vat_support.sql` يضيف عمود `tax` + يحسب VAT في الـ RPC
   من `center_settings.tax_rate`، وPOS يعرض الضريبة، والفاتورة المطبوعة تُظهرها.
   (مغطّى بـ 4 اختبارات). يبقى فقط ضبط النسبة 5% من شاشة الإعدادات.
2. ~~**الحجز الإلكتروني للعملاء (Online Booking)**~~ ✅ **تم التنفيذ (2026-06-28)** —
   صفحة عامة `/book` 24/7 بدون تسجيل دخول، عبر RPCs آمنة للدور `anon`
   (migration `20260628000006_public_booking.sql`).
3. **تذكيرات WhatsApp/SMS التلقائية** — هيكل فقط. المنافسون الإقليميون يجعلونها
   نقطة بيع رئيسية لتقليل الـ no-shows (حتى 70%). *(يحتاج اعتماد API منك)*

### 🟠 أولوية عالية — فجوات تنافسية واضحة
4. ~~**تعدد الفروع (Multi-location)**~~ ✅ **تم التنفيذ (2026-06-28)** —
   وضع `multi` + مبدّل فروع في الهيدر + حفظ الاختيار (العزل عبر RLS/العضوية).
5. **بوابة دفع إلكترونية** (Thawani/PayTabs/Stripe) — لا شيء. المنافسون يقبلون دفعاً
   أونلاين + عربون لحجز مؤكّد. *(يحتاج حساب تاجر منك)*
6. ~~**برنامج ولاء متدرّج** (Bronze→Gold)~~ ✅ **تم التنفيذ (2026-06-28)** —
   فئات حسب الإنفاق بخصم تلقائي عند الـ checkout (migration `20260628000005`).
7. ~~**بطاقات الهدايا (Gift Cards)**~~ ✅ **تم التنفيذ (2026-06-28)** —
   إصدار بطاقة برصيد عبر RPC آمن + استرداد الرصيد داخل الـ POS/checkout
   (migration `20260628000007_gift_cards.sql`).
8. ~~**الباقات والعروض (Packages/Bundles)**~~ ✅ **تم التنفيذ (2026-06-28)** —
   إنشاء باقات خدمات بسعر مخفّض وبيعها عبر الـ POS
   (migration `20260628000008_packages_bundles.sql`).
9. **حماية من الـ No-Show** (عربون/رسوم إلغاء) — غير موجودة؛ ميزة بارزة لدى Booksy/StyleSeat.

### 🟡 أولوية متوسطة — تميّز إضافي
8. **التسويق** (حملات Email/SMS، عروض مستهدفة، تقييمات/مراجعات) — غائب.
9. **معرض قبل/بعد + ملفات خدمة بالصور** — غائب (مهم لصالونات التجميل).
10. **تطبيق جوال مخصص للعميل / بوابة عميل** — لدينا PWA إداري فقط، لا بوابة عميل.
11. **AI** (مستقبل المنافسة 2026: روبوت حجز، توصيات) — غائب.
12. **تكامل محاسبي** (QuickBooks/تصدير) — تصدير JSON فقط.

## 4) ✅ نقاط قوتنا التنافسية (نحافظ عليها)
- **عربي RTL أصيل + عملة OMR** مدمجان من الأساس (كثير من العالميين ضعيف هنا).
- **سعر لمرة واحدة** (تسليم مملوك) مقابل اشتراكات شهرية متكررة للمنافسين —
  ميزة سعرية قوية للسوق المحلي.
- **PWA يعمل offline + قابل للتثبيت** بدون رسوم متجر تطبيقات.
- **معمارية نظيفة + كود عالي الجودة** (strict TS، 106 اختبار) — أساس متين للتوسّع.
- **خصوصية البيانات**: بيانات العميل في مشروعه الخاص (Supabase) لا في marketplace.
